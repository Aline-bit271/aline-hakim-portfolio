Project Title: GoTogether ‚Äì A Carpooling Application for Lebanon

Objective:
To reduce transportation costs and traffic in Lebanon by connecting drivers and passengers through a digital ride-sharing platform.

Tools & Stack:
- Flutter (Frontend)
- .NET Core Web API (Backend)
- SQL Server (Database)

System Overview:
GoTogether allows users to register as drivers or passengers. Drivers can offer rides and passengers can search and book them.

Backend Functionality:
- API endpoints for login, ride creation, booking, rating
- Secure authentication using JWT tokens

Database Tables:
- Users: ID, Name, Email, PasswordHash, Role
- Rides: ID, From, To, Date, Time, Seats, DriverID
- Bookings: ID, RideID, UserID, Status
- Ratings: ID, ReviewerID, RevieweeID, Score, Comment

Conclusion:
GoTogether improves commute sharing, cuts fuel costs, and is scalable for national rollout. Future work includes chat features and payment integration.
