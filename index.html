Major Project – GoTogether App

Overview

GoTogether is a full-stack mobile app for carpooling in Lebanon. It connects passengers and drivers based on routes and times, reducing traffic and transport cost.

Technologies: Flutter (frontend), .NET Core Web API (backend), SQL Server (database)

Features

  • User registration & login
  • Post & search for rides
  • Book seats
  • Live ride status
  • Ratings and reviews

Database Design

  • Users: ID, Name, Email, Role (driver/passenger)
  • Rides: From, To, Time, Available Seats
  • Bookings: UserID, RideID

Future Improvements

  • In-app messaging
  • Payment integration
  • Enhanced maps support
